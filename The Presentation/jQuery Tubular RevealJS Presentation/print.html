The JavaScript


$(document).ready(function() {
                $('#wrapper').tubular({videoId: 'idOfYourVideo'});
        });

A few things to consider

  1. Better suited for a Desktop Site.
  2. If your YouTube Video has advertisements they can be distracting.
  3. In your design use CSS3 Transparency to allow for the background to be seen behind the page content.
